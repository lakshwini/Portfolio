Cartographie des Cinémas en France

Projet de data engineering : scraper le catalogue AlloCiné pour extraire 592 cinémas français, les géolocaliser, et créer une cartographie interactive explorant 2948 films.

Les données

Structure JSON

Les données sont organisées en trois fichiers JSON distincts, du plus simple au plus détaillé, permettant une intégration flexible avec Leaflet.js et Folium.

📌 cinemas_simple.json
{ "cinemas": [ { "nom": "L'Amphi", "ville": "Bourg-en-Bresse", "dept": "Ain", "lat": 46.2045, "lon": 5.2245 }, ... ] }

Fichier léger pour cartographie basique. Contient nom, localisation et géocoordonnées des 592 cinémas.

📋 cinemas_adresses.json
{ "cinemas": [ { "nom": "L'Amphi", "adresse": "1 allée Brotteaux", "code_postal": "01000", "ville": "Bourg-en-Bresse", "lat": 46.2045, "lon": 5.2245, "url_allocine": "..." }, ... ] }

Fichier enrichi avec adresses complètes et URLs AlloCiné pour navigation directe.

🎬 cinemas_films.json
{ "cinemas": [ { "nom": "L'Amphi", "lat": 46.2045, "lon": 5.2245, "films": [ "La Vénus électrique", "Michael", "Mortal Kombat II", ... ] }, ... ] }

Fichier complet incluant le catalogue filmographique de chaque cinéma (2948 films total).

Ce que j'ai appris

Compétences développées

🕷️
Web Scraping
Extraction automatisée du catalogue AlloCiné avec BeautifulSoup, gestion des erreurs et pagination.
📍
Mashup de données
Enrichissement des adresses brutes avec géolocalisation via l'API data.gouv.fr (99.8% de succès).
🗺️
Cartographie interactive
Création de cartes Leaflet.js/Folium explorant cinémas par région, films par établissement.
💾
Persistance & sérialisation
Structuration et validation des données en JSON, maintien de la cohérence sur 592 cinémas et 2948 films.

Comment ça marche

Fonctionnalités principales

1️⃣
Scraper AlloCiné
BeautifulSoup parse les pages cinéma, extrait noms, adresses et listes de films.
2️⃣
Enrichir avec géolocalisation
Les adresses sont converties en latitude/longitude via l'API data.gouv.fr.
3️⃣
Exporter en JSON
Trois fichiers JSON distincts pour différents cas d'usage (simple, enrichi, avec films).
4️⃣
Visualiser sur carte
Leaflet.js affiche cinémas comme marqueurs, explorable par région et par catalogue.
