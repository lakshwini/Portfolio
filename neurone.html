L'idée

Qu'est-ce qu'un perceptron ?

J'ai programmé de zéro un neurone artificiel capable de décider si une couleur est rouge ou non, sans utiliser aucune bibliothèque de machine learning.

Une couleur c'est trois chiffres : rouge, vert, bleu, chacun entre 0 et 1. L'objectif du neurone est de trouver la frontière dans cet espace qui sépare les rouges des non-rouges.

Tout est codé en Python pur — la logique d'apprentissage, les corrections de poids, la mesure de convergence : tout est écrit ligne par ligne.

Initialisation
État initial p = [1, 1, 1], pas d'apprentissage ε = 0.2
Époque
Présentation des 15 couleurs de la liste d'entraînement, une par une
Correction
Si erreur : pᵢ ← pᵢ ± ε·eᵢ selon le type d'erreur
Convergence
Répétition jusqu'à ce que l'écart entre deux époques soit inférieur à 0.0001

Le code

Ce que j'ai écrit

# neurone.py
def activation(p, e):
    somme = sum([p[i]*e[i] for i in range(len(p))])
    return somme >= 1

def apprentissage(p, e, b):
    alpha = 0.1
    sortie = activation(p, e)
    erreur = b - sortie
    for i in range(len(p)):
        p[i] += alpha * erreur * e[i]
    return p

def ecart(p1, p2):
    return sum((p2[i]-p1[i])**2 for i in range(len(p1)))
